# 🏠 California House Price Prediction App :: https://xgboost-fr3rztpfgy78qqdppmwyqu.streamlit.app/

This project is a Machine Learning web application built using **Streamlit** that predicts house prices based on housing features such as income, house age, location, and population.

The model used for prediction is **XGBoost Regressor**.

---

# 🚀 Project Overview

The application allows users to input housing details and get an estimated house price instantly using a trained machine learning model.

---

# 🧠 Machine Learning Model

Model Used:
- XGBoost Regressor

Why XGBoost?
- High prediction accuracy
- Handles large datasets efficiently
- Works well with regression problems

---

# 📊 Dataset

Dataset Used:
- California Housing Dataset

Features used in the model:

1. MedInc – Median income in block
2. HouseAge – Median house age
3. AveRooms – Average number of rooms
4. AveBedrms – Average number of bedrooms
5. Population – Block population
6. AveOccup – Average house occupancy
7. Latitude – Geographic latitude
8. Longitude – Geographic longitude

---

# 🖥️ Streamlit Web Application

The Streamlit app allows users to:

- Enter housing feature values
- Predict house prices
- Interact with the ML model through a web interface
